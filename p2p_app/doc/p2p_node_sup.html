start_mst_worker(SupRef::pid(), Name::term()) -> {ok, Pid::pid()} | {error, {Reason::term(), Pid::pid()}}

SupRef: The reference to the supervisor where the worker process will be started.
Name: A term specifying the name for the worker process.

returns: {ok, Pid} if the worker process is successfully started, where Pid is the process identifier of the new worker. {error, {Reason, Pid}} if the worker cannot be started, where Reason is a term specifying the reason for the failure, and Pid is the process identifier of the worker process.

Starts a worker process for the Minimum Spanning Tree (MST) computation. This function is typically used within a supervisor to start a worker process.
